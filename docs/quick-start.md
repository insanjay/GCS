# Quick Start Guide

This guide helps you start using GCS immediately.

---

## Start the Application

Run:

```bash
python main.py
```

You will see:

* Webcam window
* Visual HUD overlay
* Status information

---

## Basic Gestures

### Move Cursor

Raise your index finger.

Move your hand to control cursor.

---

### Left Click

Pinch:

Thumb + Index finger

---

### Right Click

Pinch:

Thumb + Middle finger

---

### Scroll

Raise:

Index + Middle fingers

Move hand up/down.

---

### Pause

Make a fist.

System pauses for safety.

---

### Exit
For exit press **ctrl + c** in terminal

#### Note: Currently working on press Q to exit

---

## Tips for Best Performance

* Use good lighting
* Keep hand clearly visible
* Avoid fast sudden movements
* Stay within camera frame
