# Gesture Reference

This document describes supported gestures.

---

## Cursor Movement

Gesture:

Index finger raised

Action:

Moves cursor

---

## Left Click

Gesture:

Thumb + Index pinch

Action:

Left click

---

## Right Click

Gesture:

Thumb + Middle pinch

Action:

Right click

---

## Scroll Mode

Gesture:

Index + Middle fingers raised

Action:

Scroll mode activated

Move hand up/down.

---

## Pause Mode

Gesture:

Closed fist

Action:

Pauses system

---

## Exit

For exit press **ctrl + c** in terminal

#### Note: Currently working on press Q to exit

---

## Notes

Gestures require:

* Clear visibility
* Proper lighting
* Stable hand
