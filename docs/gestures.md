# Configuration Guide

GCS behavior can be customized using:

[config.py](/config.py)

---

## Common Settings

### Cursor Sensitivity

Controls cursor speed.

Higher value = faster movement

---

### Smoothing Factor

Controls cursor stability.

Higher value = smoother but slower

---

### Scroll Speed

Controls scroll sensitivity.

---

### Gesture Cooldown

Prevents repeated accidental clicks.

---

## How to Modify

Open:

config.py

Edit values.

Restart application.

---

## Recommendation

Adjust gradually for best experience.
